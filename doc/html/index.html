Ecommerce

Semplice simulazione di un sito Ecommerce. Un introduzione all'utilizzo del "sito" si può trovare nel file README.md
Il codice è liberamente modificabile e scaricabile alla pagina GitHub


Le pagine principali del progetto sono:

LOGIN
ADMIN_PAGE
USER_PAGE
CARRELLO

Le classi che gestiscono e contengono le funzioni più importanti sono i gestori:

  • HandlePanel:
    Gestore dei pannelli.
  • HandleProduct:
    Gestore dei Prodotti e dell'IO da file dei prodotti.
  • HandleUser:
    Gestore degli utenti e dell'IO da file degli utenti.
  • HandleCarrello:
    Gestore del carrello e dei relativi acquisti.


Il tutto è spiegato meglio graficamente attraverso gli UML:
